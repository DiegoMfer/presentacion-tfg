What is Zarr?
Zarr is a project to store tensors
Zarr is a project to store datasets bigger than RAM by loading only necessary chunks of information
What is RDF?
Resource Description Framework (RDF) is a general framework for representing interconnected data on the web.
RDF stores data in the form of triples

(Subject, Predicate, Object)

Data dump example in ntriples format:
